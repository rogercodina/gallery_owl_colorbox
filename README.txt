Gallery Owl Caroussel with colorbox
===================================

Adds colorbox behaviour to Owl Caroussel galleries with specific class "owl-colorbox". So edit your Owl Caroussel module's profile and add the class "owl-colorbox" on "Base class" field so that galleries gets colorbox behaviour. Also make sure your owl carousel is in a field named "field_imatges" in a "Basic page".

This module is no longer necessary, just us the patch in this issue: https://www.drupal.org/node/2307421
